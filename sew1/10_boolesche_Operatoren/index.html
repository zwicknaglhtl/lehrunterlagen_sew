Logische Ausdrücke

  • Kompliziertere Bedingungen können oft nicht mit einem einfachen Vergleich ausgedrückt werden.
    • Man kann z.B. nicht 3 < a > 5 schreiben.
  • Logische Ausdrücke verknüpfen Wahrheitswerte mit logischen (booleschen) Operatoren (UND, ODER, Negation) und liefern als Ergebnis wieder einen Wahrheitswert.
  • Beispiele in Alltagssprache:
    • Wenn die Note größer gleich 1 UND kleiner gleich 5 ist , dann ist die Schulnote gültig.
    • Wenn die Ampelfarbe gelb ODER grün ist, dann lasse den Schranken offen.

Boolesche Operatoren

Boolesche Operatoren, geordnet nach Rangfolge der Auswertung:

Verknüpfung Ergebnis ist true, wenn: Allgemeine Form in Java
UND alle Wahrheitswerte true sind Wahrheitswert1 && Wahrheitswert2
ODER mindestens ein Wahrheitswert true ist Wahrheitswert1 || Wahrheitswert2
Negation der Wahrheitswert false ist !Wahrheitswert1

Auch bei logischen Ausdrücken kann man runde Klammern ( ) setzen, um die Reihenfolge der Auswertung zu bestimmen.

Es gilt als guter Programmierstil, bei komplexeren logischen Ausdrücken immer Klammern für ein einfacheres Verständnis zu setzen.

Gleichwertige Ausdrücke

Der linke und rechte logische Ausdruck sind jeweils gleichwertig.

jeweils gleichwertig
!(a == b) a != b
!(a < b) a >= b
!(a > b) a <= b
x && true x
x || true true
!! x x
jeweils gleichwertig
!(a != b) a == b
!(a <= b) a > b
!( a>= b) a < b
x && false false
x || false x
