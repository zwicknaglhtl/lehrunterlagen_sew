Integer-"Falle"

  • Eigentlich würden wir uns im folgenden Beispiel als Ergebnis 2.25 erwarten. 
    
			float ergebnis = 9 / 4;	//Ergebnis ist 2.0
  • Wenn alle Werte eines Ausdrucks Integer sind, ist das Ergebnis ebenfalls ein Integer!
  • Erst wenn einer der Werte ein float (bzw. ein anderer Datentyp für Kommazahlen) ist, wird das Gesamtergebnis ebenfalls als float berechnet. 
    
			//Ergebnis ist bei den folgenden Ausdrücken immer 2.25.
			float ergebnis = 9f / 4; //Durch das f wird die Zahl 9 als float gespeichert
			ergebnis = 9.0 / 4; //Durch die Kommastelle wird die Zahl 9 als float gespeichert.

Integer-"Falle"

Man kann auch einen Operanden mit 1.0 multiplizieren. Dadurch bringt man ebenfalls einen float in die Rechnung ein, ohne das Ergebnis zu verändern.

int x = 9; int y = 4; float ergebnis = x * 1.0 / y; //ergebnis ist 2.25

Shorthand Operatoren

  • Oft möchte man eine Variable um einen bestimmten Wert verändern. 
    
			int x = 150;
			x = x + 20;	//x hat jetzt den Wert 170
  • Das kann man auch kürzer schreiben (muss man aber nicht). 
    
			int x = 150;
			x +=  20;
  • Es gibt auch die Shorthand Operatoren -=, *= und /=

String (Text)

  • Möchte man einen Text im Programm verwenden, muss man den Text unter Anführungszeichen " " setzen.
  • Text kann in Variablen vom Datentyp String gespeichert werden.
  • In Texten können und sollen Umlaute (ä,ö,...) verwendet werden (nur im "englischsprachigen" Code selbst nicht)!

String vorname = "Lisa";
String begruessung = "Hallo! Schön, daß du da bist!";

Der + Operator auf Strings

Wird der + Operator auf Strings angewendet, verkettet er die Strings, d.h. er fügt sie aneinander.


String vorname = "Lisa";
String begruessung = "Hallo "+vorname+"! Schön, daß du da bist!";
print(begruessung); //Ausgabe: "Hallo Lisa! Schön, daß du da bist!"

Die + Operator "Falle"

  • Wenn ein Wert eines Ausdrucks ein String ist, werden alle Werte in Strings umgewandelt und verkettet. 
    
		print("2 + 5 = "+2+5); //Ausgabe: 2 + 5 = 25
  • Nur wenn alle Werte eines Ausdruck numerisch sind, bedeutet der + Operator eine Addition. 
    
			print("2 + 5 = "+(2+5)); //Ausgabe: 2 + 5 = 7
