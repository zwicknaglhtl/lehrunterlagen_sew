Black Box Modell

Man kann eine Funktion als "black box" betrachten.

Eine Programmiererin muss das "Innenleben" (den Code innerhalb der Funktion) nicht kennen um die Funktion zu verwenden.

Sie muss nur wissen, wie man die Funktion aufruft...

...damit die Funktion einen Output erzeugt (in unserem Fall eine grafische Ausgabe am Bildschirm).

Black Box Modell

In vielen Fällen soll eine Funktion aber nicht (nur) direkt einen Output generieren...

...sondern ein Ergebnis zurückliefern (z.B. das Ergebnis einer Berechnung).

Funktion mit Rückgabewert

Liefert eine Funktion einen Wert zurück, schreibt man anstelle von void den Datentyp des Rückgabewerts (z.B. int, float, String,.. ) vor den Funktionsnamen.

Beispiel


//diese Funktion liefert einen int-Wert zurück 
int berechneTage(int jahre){
	...
}

Wie kann eine Funktion einen Wert zurückliefern?

Schlüsselwort:

return

Beispiel


int berechneTage(int jahre){
	int tage = 365*jahre;
	//return beendet eine Funktion  
	//und gibt einen Wert zurück
	return tage;
}
	
void setup(){
	//speichert den Rückgabewert in der Var. j
	int j = berechneTage(14); 
	
	//verwendet den Rückgabewert direkt
	println(berechneTage(2)); 
}

Der zurückgelieferte Wert "ersetzt" den Aufruf der Funktion - nach deren Ausführung.

Beispiel v2


	int berechneTage(int jahre){

return kann auch das Ergebnis einer Anweisung zurückliefern.

Mehrere Return Anweisungen


int berechneTage(int jahre){

	}
	
	return 365*jahre;
}

Return ohne Rückgabewert


			void printGewicht(int gewicht) {
				if (gewicht < 0) {
					return;
				}
				println("Dein Paket wiegt "+gewicht+ "kg.");
			}
			
			void setup (){
			   printGewicht(10); //Ausgabe: "Dein Paket wiegt 10kg."
			   printGewicht(-4); //keine Ausgabe
			}

return kann auch ohne einen Wert verwendet werden.

Der Returntyp muss dann aber void sein.

Ein return beendet immer sofort die Ausführung der Funktion.

Zusammenfassung

  • Ein return beendet immer sofort die Funktion.
  • Mittels dem Schlüsselwort return kann eine Funktion einen Wert zum Funktionsaufruf zurück liefern.
  • Im Kopf der Funktion muss der Datentyp des Rückgabewerts angegeben werden.
