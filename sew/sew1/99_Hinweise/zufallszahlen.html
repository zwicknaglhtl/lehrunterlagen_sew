Zufallszahlen erzeugen in Java

In Java kann man eine ganzzahlige Zufallszahl zwischen min und max (inklusive) mit folgendem Befehl erzeugen:


int rand = min + (int)(Math.random() * ((max - min) + 1));

Beispiel:


//erzeugt eine Zufallszahl zwischen 1 und 10 (inklusive 1 und 10)
int rand = 1 + (int)(Math.random() * ((10 - 1) + 1));

In Processing könnte man auch random() verwenden. Diese Funktion ist allerdings nur in Processing und nicht im "puren" Java vorhanden. Wir versuchen von Anfang an möglichst viele Java Funktionalitäten zu verwenden, damit später der Umstieg leichter fällt.

Sobald wir wissen wie Funktionen funktionieren, können wir in unseren Programmen auch eine einfache Hilfsfunktion erstellen, die uns das Erzeugen von Zufallszahlen erleichert.


  int random(int min, int max) {
    ...
  }

Erklärung

Die Java Math Funktion Math.random() gibt einen double Wert im Bereich [0, 1) zurück, wobei die 1 nicht mit eingeschlossen ist.


double rand = Math.random();      //liefert eine Zufallszahl als Dezimalzahl zwischen 0 und 1 (exklusive)

Multiplizieren wir die zurückgelieferte Zufallszahl mit einer beliebigen Zahl x erhalten wir eine Zufallszahl im Bereich [0,x) (0 bis x, x exklusive).


double rand = Math.random() * 5;  //liefert eine Zufallszahl zwischen 0 und 4.999...

Multiplizieren wir die zurückgelieferte Zufallszahl mit max-min erhalten wir eine Zufallszahl im Bereich [0,max-min).


int min = 1;
int max = 10;

double rand = Math.random() * (max-min); //erzeugt eine Zufallszahl zwischen 0 und 8.9999...

Da die kleinste mögliche Zufallszahl min sein soll, müssen wir min noch hinzuaddieren.


  int min = 1;
  int max = 10;
  
  double rand = min + (Math.random() * (max-min)); //erzeugt eine Zufallszahl zwischen 1 und 9.9999...

Die größte mögliche Zufallszahl ist noch immer nicht inkludiert. Wir müssen also noch 1 zum Wertebreich hinzuaddieren.


  int min = 1;
  int max = 10;
  
  double rand = min + (Math.random() * (max-min+1)); //erzeugt eine Zufallszahl zwischen 1 und 10.9999...

Abschließend müssen noch alle Nachkommastellen entfernt werden, um ein ganzahliges Ergebnis zu erhalten. Dies geschieht mit einem sogenannten Cast in einen Integer mittels (int).


  int min = 1;
  int max = 10;
  
  int rand = min + (int) (Math.random() * (max-min+1)); //erzeugt eine Zufallszahl zwischen 1 und 10
