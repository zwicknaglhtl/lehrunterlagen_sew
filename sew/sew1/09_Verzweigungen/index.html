Verzweigungen

Softwareentwicklung 1

Wozu Verzweigungen?

  • Bis jetzt werden alle Anweisungen unserer Programme einfach der Reihe nach ausgeführt.
  • In vielen Anwendungsfällen sollen Codeblöcke nur unter bestimmten Bedingungen ausgeführt werden.

Bedingungen

Bedingung

  • Eine Bedingung ist eine "Frage", die man beim Programmieren formuliert.
  • Die Antwort wird erst zur Laufzeit (während der Ausführung des Programms) berechnet.
  • Eine Bedingung hat als Antwort immer einen Wahrheitswert, also nur zwei mögliche Antworten:
    • true, wenn die Antwort "Ja" ist (Bedingung trifft zu).
    • false, wenn die Antwort "Nein" ist (Bedingung trifft nicht zu).
  • Beispiele aus der Alltagssprache:
    • Wiegt der Sack Erdäpfel mehr als 3kg?
    • Ist das Fenster offen?

Formulieren von Bedingungen

Auf den nächsten Seiten werden folgende Arten von Bedingungen näher beschrieben:

  • Vergleich von Zahlenwerten
  • Vergleich von Strings
  • Boolesche Variablen

Vergleich von Zahlenwerten (1/3)

Der Vergleich gibt an, ob ein bestimmtes mathematisches Größenverhältnis zwischen zwei Zahlenwerten zutrifft (true) oder nicht (false).

Allgemeine Form eines Vergleichs zwischen Zahlenwerten:

Zahlenausdruck1 Vergleichsoperator Zahlenausdruck2

Vergleich von Zahlenwerten (2/3)

Zahlenausdruck1 Vergleichsoperator Zahlenausdruck2

Vergleichsoperatoren

Beschreibung mathematisches Symbol Java-Vergleichsoperator
gleich = ==
ungleich ≠ !=
kleiner < <
kleiner oder gleich ≤ <=
größer > >
größer oder gleich ≥ >=

Vergleich von Zahlenwerten (2/3)

Zahlenausdruck1 Vergleichsoperator Zahlenausdruck2

Beispiele


int a = 2, b = 3;

a == 2;	//ist true

a != 2; //ist false

a >= b; //ist false

a*2 < b+2; //ist true

Wichtige Hinweise

  • Beachte den Unterschied zwischen Vergleich und Zuweisung!

    =

    Zuweisung

    
						int x=3; //x wird der 3 zugewiesen

    ==

    Vergleich

    
					x==3; //ist true
  • Vergleiche dürfen nur zwei Werte enthalten.
    • Folgender Ausdruck ist daher z.B. ungültig: 
      1 <= a < 3
    • Siehe boolesche Operatoren um solche Bedingungen ausdrücken zu können.

Vergleich von Strings (1/2)

Gibt an, ob zwei String-Ausdrücke gleich (true) oder ungleich (false) sind.

Dabei wird zwischen Groß- und Kleinschreibung unterschieden.

Allgemeine Form eines Vergleichs zweier Strings:

string1.equals(string2)

Achtung! Es ist zwar möglich Vergleich von Strings mit == und != zu schreiben. Diese Ausdrücke liefern aber nicht das gewünschte Ergebnis

string1 == string2

Vergleich von Strings (2/2)

string1.equals(string2)

Beispiele


String x = "abc", y = "HTL";

x.equals("ABC") 	// ist false
"abc".equals(x)  	// ist true
x.equals(y) 		// ist false
y.equals("HT"+"L")	//ist true

boolesche Variablen

  • Ein Wahrheitswert (true oder false) kann auch in einer Variable gespeichert werden.
  • Eine solche Variable besitzt den (neuen) Datentyp boolean.
    • Ein boolean kennt nur zwei Werte - true oder false.
  • Weiters gibt es die logischen Konstanten true und false. 

int a = 2, b = 3;

boolean isOpen = true;		//boolesche Variable mit true als Anfangswert

boolean aGroesserB;
aGroesserB = a > b;		//false wird zugewiesen

boolean bUngleichA = false;
bUngleichA = (b != a);		//true wird zugewiesen

Verzweigung

Die if-Anweisung

if-Anweisung - Syntax

Wenn die Bedingung in den runden Klammern() erfüllt (true) ist, werden die Anweisungen zwischen den geschwungenen Klammern ausgeführt.

Wenn die Bedingung nicht erfüllt (false ) ist, werden diese Anweisungen übersprungen. 


						if (Bedingung) {
							//Code innerhalb des Blocks wird nur ausgeführt, 
							//wenn die Bedingung wahr ist
							Anweisung 1;
							Anweisung 2;
							...
						}

if-Anweisung - Beispiel 1


	int x = 350;
	int speed = 5;
	
	if (x>400) {
	    speed = 3;
	}
	
	println("Speed ist "+speed); //Ausgabe: Speed ist 5   

if-Anweisung - Beispiel 2


		int x = 350;
		int speed = 5;
		
		if (x<=350) {
		    speed = 3;
		}
		
		println("Speed ist "+speed); //Ausgabe: Speed ist 3   

Hinweise

  • Nach dem logischen Ausdruck in runden Klammern () folgt kein Strichpunkt! 
    
								if (x<=350); {
									speed = 3;
								}
  • Bei einer einzelnen Anweisung darf man die geschwungenen Klammern {} zwar weglassen, dies gilt aber als schlechter Programmierstil.

Die if-else Anweisung

if-else-Anweisung - Syntax

Eine if/else-Anweisung ist eine Erweiterung der if-Anweisung.

Wenn die Bedingung erfüllt (true ) ist, werden die Anweisungen im if-Zweig ausgeführt.

Wenn die Bedingung nicht erfüllt (false ) ist, werden die Anweisungen im else-Zweig ausgeführt.


						if (Bedingung) {
							// Anweisungen im if-­Zweig 
							Anweisung1;
							Anweisung2;
							...
						} else {
							// Anweisungen im else­-Zweig 
							Anweisung1;
							Anweisung2;
							...
						}

if-else-Anweisung - Beispiel 1


int x = 350;
int speed = 7;
 
if (x<500) {
  speed = 5;
} else {
  speed = 3;
}

println("Speed ist "+speed);   //Ausgabe: Speed ist 5

if-else-Anweisung - Beispiel 2

if - else if - else - Anweisung

if- else if - else Anweisung - Syntax

Eine if - else if - else Anweisung ist eine ununterbrochene Folge von if/else-Anweisungen.

Nur die Anweisungen im ersten if-Zweig, dessen Bedingung erfüllt ist, werden ausgeführt.

Ist keine Bedingung erfüllt, werden die Anweisungen im letzten else-Zweig ausgeführt, falls vorhanden.


					if ( Bedingung1 ) {
						// Anweisungen, wenn Bedingung1 erfüllt 
						...
					} else if ( Bedingung2 ) {
						// Anweisungen, wenn Bedingung2 erfüllt (und Bedingung1 nicht) 
						...
					} 
					//...optional weitere else if Anweisungen
					else {
						// Anweisungen, wenn keine Bedingung erfüllt 
						// kann auch weggelassen werden
						...
					}

int x = 100;

Verschachtelungen

Innerhalb jedes Blocks können beliebige Anweisungen stehen. Somit können innerhalb der Blöcke eines if-Statements weitere if-Statements, Funktionsaufrufe,... stehen.


int x = PopUp.readInt("Gib eine beliebige Zahl ein!");
if(x < 100) {
	println("Hmm, deine Zahl ist kleiner als 100.");
	
	if (x < 50) {
		println("Ich weiß auch, dass sie kleiner als 50 ist!");
	} else {
		println("Deine Zahl liegt zwischen 50 und 100.");
	}
}

switch-Anweisung

switch-Anweisung - Syntax

Bei der switch-Anweisung bestimmt ein String oder ein ganzzahliger Ausdruck den Zweig, der ausgeführt wird.

Jeder case-Block muss mit einem break beendet werden (ansonsten würde - unabhängig von den Werten bei case - der Code bis zum nächsten break ausgeführt werden.) 


					switch ( String oder ganzzahligerAusdruck ) { 
						case Wert1:
							// Anweisungen, wenn der Ausdruck den Wert1 hat
							...
							break;
						case Wert2:
							// Anweisungen, wenn der Ausdruck den Wert2 hat ...
							break;
						
						... //weitere case-Statements
						
						default:	//kann man auch weglassen
							//Anweisungen, falls kein passender Wert
					}

switch-Anweisung - Beispiel 1


int note;
...

switch (note) {
	case 1:
		println("Sehr gut");
		break;
	case 2:
		println("Gut");
		break;
	case 3:
		println("Befriedigend");
		break;
	case 4:
		println("Genügend");
		break;
	case 5:
		println("Nicht genügend");	
		break;
	default:
		println("*** Hier stimmt etwas nicht! ***");
		break;
}

Boolean als Rückgabetyp (1/2)

Beispiel für eine Funktion die einen Wert mit dem Datentyp boolean zurückliefert.


boolean kochtdasWasser(float temperatur){
	if(temperatur >= 100) {
		return true;
	} else {
		return false;
	}
}

void setup() {
	float wasserTemperatur = 62.5;

	if(kochtdasWasser(wasserTemperatur) == true) {
		println("Gib die Nudeln in den Topf!");
	}
}

Boolean als Rückgabetyp (2/2)

Schauen wir uns folgenden Code von der letzten Folie genauer an.


	...
		if(kochtdasWasser(wasserTemperatur) == true) {
			println("Gib die Nudeln in den Topf!");
		}
	...

Die Bedingung ist true wenn der Rückgabewert der Funktion true ist. Dies kann auch kürzer geschrieben werden!


	...
		if(kochtdasWasser(wasserTemperatur)) {
			println("Gib die Nudeln in den Topf!");
		}
	...

return und Verzweigungen

Wenn eine Funktion einen Wert zurückliefert (Rückgabetyp ist nicht void), muss in jedem möglichen Fall einen Wert zurück geliefert werden.


		String doSomething(int a) {
			if (a>=10) {
				return "zweistellige Zahl";
			}
			//Fehler, da nicht in jedem Fall ein Wert zurückgegeben wird
		}
		

		String doSomething(int a) {
			if (a>=10) {
				return "zweistellige Zahl";
			}

			return "einstellige Zahl";	//korrekt. Egal ob die Bedingung zutrifft oder nicht, 
			                            //es wird immer ein Wert zurückgegeben
		}
