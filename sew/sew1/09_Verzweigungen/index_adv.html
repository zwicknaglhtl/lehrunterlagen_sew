switch-Anweisung - Syntax

Bei der switch-Anweisung bestimmt ein String oder ein ganzzahliger Ausdruck den Zweig, der ausgeführt wird.

Jeder case-Block muss mit einem break beendet werden (ansonsten würde - unabhängig von den Werten bei case - der Code bis zum nächsten break ausgeführt werden.) 


					switch ( String oder ganzzahligerAusdruck ) { 
						case Wert1:
							// Anweisungen, wenn der Ausdruck den Wert1 hat
							...
							break;
						case Wert2:
							// Anweisungen, wenn der Ausdruck den Wert2 hat ...
							break;
						
						... //weitere case-Statements
						
						default:	//kann man auch weglassen
							//Anweisungen, falls kein passender Wert
					}

switch-Anweisung - Beispiel 1


int note;
...

switch (note) {
	case 1:
		println("Sehr gut");
		break;
	case 2:
		println("Gut");
		break;
	case 3:
		println("Befriedigend");
		break;
	case 4:
		println("Genügend");
		break;
	case 5:
		println("Nicht genügend");	
		break;
	default:
		println("*** Hier stimmt etwas nicht! ***");
		break;
}
