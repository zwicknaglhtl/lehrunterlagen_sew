Schleifen

Schleifen ermöglichen ein Stück Code zu wiederholen, solange eine Bedingung wahr ist.

3 Arten von Schleifen

  • Kopfgesteuerte Schleife, in der Fachsprache auch while-Schleife genannt
  • Fußgesteuerte Schleife, in der Fachsprache auch do-while-Schleife genannt
  • Zählschleife, in der Fachsprache for-Schleife genannt

while-Schleife (kopfgesteuerte Schleife)

  • Eine while (deutsch: "solange") Schleife wiederholt einen Codeblock solange die Bedingung wahr ist.

Allgemeine Form einer while-Schleife:


				while (Bedingung) {
					
					    //wird ausgeführt, solange die Bedingung wahr ist
				
				}

Beachte: Wie bei der if-Anweisung steht die Bedingung wieder in runden Klammern (), gefolgt von einem Codeblock in geschwungenen Klammern {}.

while-Schleife: Beispiel 1


		String message = "Schleifen sind genial!";
		int y = 40;
		while (y < 400) {
			text(message, 30, y);
			y += 30;
		}

Die Bedingung muss irgendwann false werden, sonst läuft die Schleife für immer.

while-Schleife: Beispiel 2


		String message = "Schleifen sind genial!";
		int y = 500;
		while (y < 400) {
			text(message, 30, y);
			y += 30;
		}

Wenn die Bedingung gleich zu Beginn false ist, wird die Schleife nie ausgeführt.

while-Schleife: Beispiel 3


int i = 0;
while (i < 2) {
    println("i hat den Wert "+i);
    i = i + 1;
}

Ausgabe:


i hat den Wert 0
i hat den Wert 1

Schleifen Design

Bei jeder Schleife müssen folgende drei Fragen beantwortet werden:

  • Was möchte ich wiederholen?
  • Was möchte ich bei jedem Durchlauf ändern?
    • ...auch damit die Bedingung irgendwann falsch wird.
  • Wie lange soll der Code wiederholt werden?

Schleifen Design - Beispiel


		String message = "Schleifen sind WIRKLICH genial!";
		int y = 40;
		while (y < 400) {
			text(message, 30, y);
			y += 30;
		}

do-while-Schleife (fußgesteuerte Schleife)

  • Bei einer do-while Schleife steht die Bedingung am Ende (=fußgesteuert).
  • Eine do-while Schleife wird auf jeden Fall mindestens einmal ausgeführt (im Gegensatz zur while-Schleife)

Allgemeine Form einer do-while-Schleife:


	do  {
		
		//wird ausgeführt, solange die Bedingung wahr ist

	} while (Bedingung);

do-while-Schleife - Beispiel


String message = "Schleifen sind genial!";
int y = 40;

do  {
	text(message, 30, y);
	y += 30;
} while (y < 400);

do-while-Schleife - Beispiel 2


int i = 0;

do {
	println("i hat den Wert "+i);
	i = i + 1;
}  while (i < 2);

Ausgabe:


i hat den Wert 0
i hat den Wert 1

for-Schleife (Zählschleife)

  • Eine for-Schleife, auch Zählschleife genannt, wird gerne eingesetzt wenn die Anzahl der Wiederholungen zu Beginn der Schleife bereits fest steht.
  • Jede while-Schleife kann auch als for-Schleife geschrieben werden und umgekehrt. Je nach Anwendungsfall ist die eine oder andere Form leichter lesbar.

Allgemeine Form einer for-Schleife:


				for (Initialisierung; Bedingung; Veränderung ) { 
					// wird ausgeführt solange die Bedingung wahr ist
					Anweisung; 
					Anweisung; 
					...
				}

for-Schleife - Beispiel


					for (int i=0; i < 2; i = i+1) {
						    println("i hat den Wert "+i);
					}

Achtung! Die einzelnen Teile in den runden Klammern werden durch Strichpunkte ; getrennt.

Ausgabe:


i hat den Wert 0
i hat den Wert 1

for-Schleife - Programmfluss


		for (int i=0; i < 2; i = i+1) {
			    println("i hat den Wert "+i);
		}

for-Schleife - weitere Beispiele

  • Zählschleife, die hinunter zählt: 
     
			for (int i = 5; i >= 0; i = i - 1) {
				println("Countdown: " + i);
			}
  • Zählschleife, die einen Linienfächer zeichnet: 
     
			for (int i = 0; i <= 100; i = i + 10) {
				line(0, 0, 100, i);
			}
  • Zählschleife, die Zweierpotenzen berechnet: 
    
	for (int i = 1; i <= 1024; i = i * 2) {
		println(i);
	}

Endlosschleife

Eine Schleife, bei der die Bedingung immer erfüllt ist und die daher nie beendet wird, nennt man Endlosschleife.

  • Entweder ein Programmierfehler...
  • ...in manchen Fällen aber beabsichtigt!

Beispiele für (wahrscheinlich) beabsichtigte Endlosschleifen:


			while (true) { ... }
		
			for (;;) { ... }

Schleifen vorzeitig beenden

Manchmal kann man nur innerhalb eines Schleifendurchlaufs entscheiden, ob die Schleife wiederholt werden soll.

Eine Schleife kann beendet werden...

  • ...mit einer break-Anweisung
  • ...mit einer return-Anweisung, wobei dabei nicht nur die Schleife, sondern auch die Funktion beendet wird.

while-Schleife vorzeitig beenden

Allgemeine Form einer vorzeitig beendeten while-Schleife:


	while (true) { 
		Anweisung;
		...
		if ( Bedingung ) {
			break; 
		}
		Anweisung;
		... 
	}

Beispiel: Benutzereingabe, bis "exit" eingegeben wird


while (true) {
	String name = PopUp.readLine("Ein Name ('exit' zum Beenden):");
	if (name.equals("exit")) {
		break;
	}
	console.log("Hallo " + name + "!");
}

Zusammenfassung

  • Schleifen ermöglichen Anweisungen im Block { ... } wiederholt auszuführen.
  • Es gibt die while-Schleife (kopfgesteuert), die do-while-Schleife (fußgesteuert) und die for-Schleife (Zählschleife)
  • Mit einer break-Anweisung kann ein Schleife vorzeitig beendet werden.
  • 
	while ( Bedingung ) {
		// ... Nur solange Bedingung true. 
	}
	
  • 
	do {
		// ... 1x immer. Dann solange Bedingung true.
	} while ( Bedingung )
	
  • 
	for ( int x = 0; x < 10; x++ ) {  // ACHTE auf den ";" als Teiler!
		// ... solange, bis Bedingung: "x < 10" zutrifft 
	}
