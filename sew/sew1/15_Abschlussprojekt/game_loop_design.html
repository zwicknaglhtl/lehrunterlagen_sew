Konsolen-Spiele mit Funktionen

z.B. 4 Gewinnt, Schiffe versenken

rein prozedural

Hinweise

  • Diese Unterlagen sollen nur einen grundsätzlichen Aufbau zeigen.
  • Es sind im konkreten Fall vielleicht mehr/andere/weniger Funktionen sinnvoll.
  • Ähnliche Überlegungen für andere Programme (z.B. eine Einkaufsliste verwalten).
  • Die Bsp. zeigen Pseudocode (Idee, kein konkreter Code!)

1. Spielfeld und Zustand

  • 2D-Array zur Darstellung des Spielfelds (board)
  • Zusätzliche Daten: Spieler, Schiffe, etc.
  • Über Parameter (Referenz/Pointer) an Funktionen übergeben

2. Anzeige

  • drawBoard(board): Zeichnet das Spielfeld
  • drawStatus(player): Zeigt Statusinfos

3. Eingabe & Validierung

  • getInput(): Eingabe holen
  • validate(input, board): Zulässigkeit prüfen

4. Zustand aktualisieren

  • applyMove(board, player, move): Spielzug eintragen
  • evt. Daten (z.B. Punkte) aktualisieren

5. Gewinn/Ende prüfen

  • checkWin(board, player): Siegbedingungen prüfen

6. Hauptschleife

initialize(board)
currentPlayer = 1
while true:
  drawBoard(board)
  move = getInput(...)
  if not validate(...): continue
  applyMove(...)
  if checkWin(...): break
  currentPlayer = nextPlayer

7. Umsetzungsschritte

  1. Grundgerüst + leere Funktionen
  2. Spielfeldanzeige
  3. Eingabe + Validierung
  4. Zug anwenden
  5. Gewinnprüfung

8. Tipps zur Modularität

  • Funktionen machen nur eine Sache.
  • Kleine Hilfsfunktionen schreiben
  • Testen mit festen Daten
